Objective

Review how to configure Adobe Acrobat Professional, and how to use it to optimize a PDF exported from a well-structured Microsoft Word or PowerPoint document.

Sections

This module has four Sections:

  • Section 1: Introduction to Optimizing PDFs
  • Section 2: Checking Accessibility
  • Section 3: Reading Order Tool
  • Section 4: Content Order and Tags Order

Each Section has a Quiz with multiple-choice questions. You should answer all Section Quiz questions correctly before moving on to the next Section.

After you have completed all of the Section Quizzes, take the Exam.

Exam

Exam 4 has ten questions that have been randomly selected from each of the Module 4 Section Quizzes.
