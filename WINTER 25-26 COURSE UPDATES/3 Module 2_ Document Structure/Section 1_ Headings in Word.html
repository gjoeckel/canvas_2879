Objectives

  • Describe how a screen reading software user navigates through a Word document with a heading structure.
  • Explain the function of main headings, major headings, and subheadings.
  • Describe the process for creating headings using the Styles gallery.
  • State the method for checking that heading text has been added to the correct position in a document's heading structure.
  • Identify the processes for modifying Heading styles.
  • Identify the process for adding a Table of Contents.
