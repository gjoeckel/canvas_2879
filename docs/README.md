# GitHub Pages Site

This directory contains the GitHub Pages site for the Canvas Course DOCX Editor.

## Setup Instructions

1. Go to your repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/docs" folder
5. Click "Save"

The site will be available at: `https://gjoeckel.github.io/canvas_2879/`

## Files

- `index.html` - Main HTML page with all course document links
- `.nojekyll` - Prevents Jekyll processing (allows raw HTML)

All links are configured to open in new tabs using `target="_blank"`.
